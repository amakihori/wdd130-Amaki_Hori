Client Report

I ended up experiencing autonomic nervous system dysfunction, and since I was there alone, my only means of communication was my roommate. My roommate was generally supportive, so we didn’t really solve any problems together, but in order to meet his criteria—which were beyond my control—I had to rely on the resources available there. Through that process, I was able to learn programming by using external materials.

When selecting images, I chose ones that were vibrant and eye‑catching for the homepage, while for the images near the navigation bar, I intentionally selected ones that were not too flashy so they wouldn’t overwhelm the user. When making changes to the grid on GitHub, I found that I couldn’t add them directly, so I ultimately decided to create the files within GitHub itself.

General Experience and Reflections

The most difficult part of this project was making subtle adjustments. I didn’t know how to approach these fine‑tuning tasks, so I ended up repeating small revisions three or four times until I achieved the result I wanted. If I could redo the project, I think I could get closer to my ideal vision from the layout stage. Through this experience, I came to understand how challenging it is to bring my ideal design to life as the project progresses.
